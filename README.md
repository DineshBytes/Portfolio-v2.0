# Settipalli Dinesh Kumar — Portfolio

Personal developer portfolio for Settipalli Dinesh Kumar — Frontend & Full Stack Developer.

## 📁 Folder Structure

```
sdk-portfolio/
├── index.html        ← Main HTML (sections, content)
├── css/
│   └── style.css     ← All styles & design tokens
├── js/
│   └── main.js       ← Scroll effects & interactivity
└── README.md
```

## ✏️ How to Edit

### Change accent color
Open `css/style.css` and find:
```css
--accent: #c8f060;
```
Replace with any color you want.

### Add a new project
Open `index.html`, find the `<!-- PROJECTS -->` section, and copy a `project-row` div and fill in your details.

### Update contact info
Search for your email/LinkedIn/GitHub links in `index.html` and update the `href` values.

## 🚀 Deploy to GitHub Pages

1. Push this folder to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://yourusername.github.io/repo-name`
